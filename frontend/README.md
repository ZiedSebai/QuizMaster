# QuizMaster - Interactive Quiz Application

A beautiful, playful quiz application built with React, TypeScript, and Tailwind CSS featuring three main views.

## Features

- **HomeView**: Enter a topic and generate a quiz
- **QuizView**: Interactive question interface with immediate feedback
- **ResultsView**: Comprehensive score display with explanations
- **Dark/Light Mode**: Theme toggle for comfortable viewing
- **Smooth Animations**: Playful transitions and micro-interactions
- **Responsive Design**: Works seamlessly on all devices

## Tech Stack

- React 18
- TypeScript
- React Router DOM
- Tailwind CSS
- shadcn/ui components
- Lucide React icons

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## Application Flow

1. **Home**: Enter any topic (currently uses static car quiz data)
2. **Quiz**: Answer 5 questions with instant feedback
3. **Results**: View your score, breakdown, and detailed explanations

## Design Highlights

- Gradient backgrounds that change per view
- Color-coded feedback (green for correct, red for incorrect)
- Progress tracking throughout the quiz
- Animated icons and smooth transitions
- Beautiful card-based layouts
